# PostgreSQL 핵심정리
## 설치
- https://www.postgresql.org/download/windows/
## 환경변수 설정
- 제어판 > 시스템 > 고급 시스템 설정
> ![ex_screen](images/psql.png)
```
$ psql --version // 버전 조회
```
## 접속
1. SQL Shell(psql)
2. 명령 프롬프트
```
$ psql -U (postgre[name]) // 접속
```
3. pgAdmin4(전용 GUI 툴)
