## CRUD
1. SQL shell 명령어

| 명령어 | 설명 | 예시코드 |
|---|:---:|---:|
\q | psql 종료 | - |
\l | 데이터베이스 조회 | - |
\c | 입력한 데이터베이스로 이동 | \c [dbName] |
\e | 외부편집기 사용 | - |
\dt | 현재 데이터베이스 테이블 확인 | - |

2. create
> ![ex_screen](images/psql2.png)
> 1) SQL shell에 차례대로 입력 `*구문 오류가 있을 시, 오류 문구가 출력됨`
> 2) \e 명령어 입력
> 3) 외부 편집기로 쿼리 수정 후 저장
> 
> ![ex_screen](images/psql3.png)
